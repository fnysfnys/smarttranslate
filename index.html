Smart Translate

Open source light weight translation software with translation memory and fuzzy matching

The Software

Smart Translate allows the user to load in .docx files and translate each sentence in a user friendly UI where reformatting is taken care of. Old translations are auto filled with translation memories. Translations can be exported to docx files with the same formatting.

  • User Friendly UI
  • Lighting Fast Response Times
  • Unlimited Custom Translation Memorys
  • Completely Open Source

Translation Memory

The software allows for creation of translation memory where every translation the user has done can be saved into and then auto-translated when the same sentence is found again. With fuzzy matching we can detect similar sentences and autotranslate aswell as autofill new numbers with the corresponding number formatting handled for you.

Open Source

All of our code is open source, meaning we allow for improvements from other keen developers like ourselves. Not only that, but Smart Translate is completely free to use, forever!

Why we beat our competitors

Smart Translate hosts the same features as big names such as SDL Trados, however our software gets rid of all the jargon and only does what translators need. This makes our program much faster and much more user friendly.
